Building Real-Time Laravel Apps with Pusher

Getting Started

In order to get started you'll need to sign up for a free account with Pusher, take a note of the credentials for your Pusher app, create a Laravel application and integrate the Pusher library into that application.

For the latter you have a few options and the Integrate Pusher section covers the options that you have.
